Tim Plummer — @Objelisks — Github — Résumé

@Pico8Bot - Link

Pico8Bot generates random gifs, similarly to how LSystemBot works, but with animation and math.

@LSystemBot - Link

LSystemBot procedurally generates short rules for Lindenmeyer systems, renders them, and posts to twitter every hour on the hour. Rendering is currently being done with Paper.js. The idea for this bot was inspired by this book.

@everypunk - Link

Everypunk is a twitter bot riff on everyword. It ran for a year and finished in 2015. Plans for more fully featured and technologically advanced rerun are in the works.

The Resistance - Link

This project is a web based remake of the board game The Resistance. I started working on this as a way to play with cryptography and peer to peer networking. Cool commutative encryption and fancy card passing protocols.

Gemsquad - Link

This is my little GLSL testbed. I've been experimenting with raymarching procedural worlds and seeing what kind of interaction can exist using only shaders. This project has gone through multiple iterations, from infinite, multiplayer grasslands to cities with portals between dimensions.

Great Dungeon in the Sky - Link

This is a little AS3 game I made back in 2010 for the TIGSource Assemblee competition. It got like 9th out of about 150? Pretty cool. Assets are not made by me.

PrimeRL - Link

Metroid Prime demake as a roguelike. Features different beams, visors, and atmospheric level generation. Made for 7drl.

Giants - Link

I made this short, barely playable game in 48 hours for Ludum Dare 33. It's got some basic collision and item collection and multiple scenes. Built using Pico-8.

Stone Stealers - Link

This is a work in progress demake of the PC game LEGO Rock Raiders. The game fits pathfinding, autotiling, and worker AI into the small Pico-8 code limit.

Picomon - Link

Another wip Pico-8 based pokemonish platformer game. Fits in 60ish monsters with unique procedural names and sounds.

Combo - Link

A javascript based game engine supporting input, enemies, items, tiles, and modular interactions between each object. Based on top of the Phaser library.

Astromancer - Link

Ludum Dare 30 game. 3D game in javascript. Involves shaders, collision, and some ui stuff.

Sticks - Link

Prototype game about exploring forests and hitting trees with sticks. Has a heightmap editor with node.js based level streaming.
